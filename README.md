# Weather Data ETL Pipeline

## Project Overview
Built an end-to-end ETL pipeline to extract real-time weather data from an API, transform it using Python, and store it in a MySQL database.

## Tech Stack
- Python
- Pandas
- MySQL
- REST API

## Features
- Extract data from OpenWeather API
- Transform data using Pandas
- Load data into MySQL database
- Perform SQL analysis

## How to Run
1. Add API key in extract.py
2. Run extract.py
3. Run load.py
